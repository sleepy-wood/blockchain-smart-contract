## Getting Started

### Prerequisites

1. npm
  ```bash
  npm i -g npm@latest
  ```
2. truffle
  ```bash
  npm install -g truffle
  ```

### Installation

1. Clone the repo
  ```bash
  git clone https://github.com/sleepy-wood/blockchain-sample.git
  ```
2. Install NPM packages
  ```bash
  npm i
  ```
3. Run App
  ```bash
  truffle compile
  ```
